Bullet Screen is a Game tracker and clock similar to apps like Lotus. By default, it will service 4 players. 

The screen will be divided horizontally and vertically with each section representing a player. Each halve will be oriented towards the edge direction of the screen (top half will be oriented to face up, bottom half oriented to face down)

All interactions with the screen are supposed to be done via touch screen and should have the corresponding visual and auditory feedbacks. All buttons should have their own sound effects to be distinct from one another and be appropriate to the action being taken (hitting the clock, changing life totals, etc)

Each Player Section will have the following components:
- Life total and buttons to change the life total
- a huge are of the section dedicated to clock management within the game
- a means of tracking commander damage from each player

Each player section will ahve two theme modes: active, and none active. An active player section will have a bright colored background and black font. An inactive player will have a dark background color and white font.

When the active player role switches between player, play a very fast animation to show the change of role between the two



Players should be positioned in a clockwise manner so that play is passed to the left

There should be 3 buttons in the middle of the screen; the center should be start-pause timer with the appropriate sound effects. The next is a menu item that opens up a menu with customization options for: time limit, number of players. The last is a button that gives the current player 10 seconds as a refund

When a player takes priority for an interaction, they should have a button that lets them return priority back to the active player.
